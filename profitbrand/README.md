# ПРОФИТБРАНД — Перформанс Маркетинг Агенция

## Бърз старт (локално)

```bash
npm install
npm start
```

Отваря се на http://localhost:3000

## Deploy на Vercel (безплатно)

1. Създай акаунт на https://vercel.com
2. Качи този проект в GitHub (ново repo)
3. В Vercel натисни "Add New Project" → избери repo-то
4. Натисни "Deploy" — готово!
5. Получаваш линк като `profitbrand.vercel.app`
6. Свържи собствен домейн от Settings → Domains

## Deploy на Netlify (алтернатива)

1. Създай акаунт на https://netlify.com
2. Плъзни папката `build/` в Netlify dashboard
3. Или свържи GitHub repo за автоматичен deploy

За production build:
```bash
npm run build
```
